# Excel Sales Dashboard

This project is an interactive sales dashboard built in Excel to analyze sales performance, customer behavior, and delivery metrics.

[Download Excel File](Project.xlsx)

[View Dashboard](Project_Dashboard.png)

## Tools Used

- Microsoft Excel
- Pivot Tables
- Charts & Dashboard

## Key Features

- Interactive filters (Year, City, Gender, Domain)
- KPI metrics (Sales, Delivery Time, Discounts)
- Sales trends over time
- Top customers and products

## Key Insights

- Identified top-performing products and customers
- Observed monthly sales trends
- Analyzed delivery performance (On-time vs Late)
- Evaluated impact of discounts on sales
